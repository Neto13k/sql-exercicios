# 📚 SQL Exercícios

Repositório com exercícios de **SQL** utilizados para prática e aprendizado de banco de dados relacionais.

O objetivo é registrar consultas e desafios resolvidos durante os estudos.

## 🧠 Conteúdos praticados

Alguns conceitos trabalhados nos exercícios:

- `SELECT`
- `WHERE`
- `ORDER BY`
- `GROUP BY`
- `JOIN`
- Funções de agregação (`COUNT`, `SUM`, `AVG`, etc.)
- Subqueries
